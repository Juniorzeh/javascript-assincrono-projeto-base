# JavaScript Assíncrono - Projeto Base

Projeto base desenvolvido para praticar conceitos de **JavaScript assíncrono**, utilizando uma estrutura simples com **HTML, CSS e JavaScript puro**.

## 📌 Objetivo

Este repositório serve como base para estudos e implementação de funcionalidades relacionadas a:

- `async / await`
- `fetch API`
- consumo de APIs
- tratamento de promessas
- manipulação assíncrona de dados no front-end

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

## 📂 Estrutura do projeto

```bash
javascript-assincrono-projeto-base/
├── css/
│   └── styles.css
├── img/
├── js/
│   └── scripts.js
└── index.html
▶️ Como executar o projeto

Como este é um projeto front-end simples, você pode executar de duas formas:

Opção 1: Abrir direto no navegador

Basta abrir o arquivo index.html no navegador.

Opção 2: Usar o VS Code com Live Server

Se estiver usando o Visual Studio Code:

Instale a extensão Live Server
Clique com o botão direito em index.html
Selecione Open with Live Server
📖 Sobre o projeto

A aplicação possui uma base visual inicial com menu lateral e estrutura preparada para evolução do projeto.
A ideia é utilizar esse template como ponto de partida para implementar funcionalidades assíncronas em JavaScript.

🚀 Possíveis melhorias

Algumas ideias para evoluir este projeto:

consumir dados de uma API pública
listar informações dinamicamente na tela
criar loading durante requisições

adicionar formulário com envio assíncrono

📷 Interface

O projeto conta com uma estrutura inicial de interface, incluindo:

menu lateral

identidade visual personalizada
